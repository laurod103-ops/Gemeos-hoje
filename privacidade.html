POLÍTICA DE PRIVACIDADE

♊ Gêmeos Hoje

Escola Esotérica · Última atualização: Janeiro de 2025

1. SOBRE ESTE APLICATIVO

O Gêmeos Hoje — Escola Esotérica é um aplicativo de horóscopo e conteúdo esotérico voltado ao signo de Gêmeos. Não realizamos cadastro de usuários nem coletamos dados pessoais diretamente.

2. DADOS COLETADOS

Este aplicativo não coleta, armazena nem compartilha dados pessoais como nome, e-mail, localização ou qualquer outra informação identificável. Todo o conteúdo é carregado localmente no seu dispositivo.

3. PUBLICIDADE (GOOGLE ADSENSE)

Este aplicativo utiliza o serviço Google AdSense para exibição de anúncios. O Google pode usar cookies e tecnologias similares para exibir anúncios personalizados com base em suas visitas anteriores a este e outros sites.

Para mais informações ou para desativar anúncios personalizados, acesse: google.com/settings/ads.

4. COOKIES

O aplicativo em si não utiliza cookies próprios. Os cookies presentes são exclusivamente do Google AdSense para fins de publicidade, conforme a política do Google.

5. INSTALAÇÃO COMO PWA

Ao instalar o aplicativo no seu dispositivo como PWA (Progressive Web App), nenhum dado adicional é coletado. O aplicativo funciona de forma independente, sem conta de usuário.

6. DIREITOS DO USUÁRIO (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), como não coletamos dados pessoais, não há dados para solicitar exclusão ou acesso. Caso tenha dúvidas, entre em contato pelo e-mail abaixo.
